## Study Rule

```
1. 주 1회 온라인 모임 (월요일 21시) -> 각자 파트 10~15분 발표
2. 각자 공부한 내용을 정리해서 깃허브에 올리기 (코드가 없어도 괜찮음)
3. 온라인 모임 불참: (가급적) 3일 전까지 미리 알려줘서 발표 날짜 변경.
4. 발표자 선정 방법: 선 지원 후 사다리타기.
```
